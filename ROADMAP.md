# BaeSQL Roadmap

## 0.1

- Custom `.bae` binary file format
- Single-file durable database
- Basic SQL parser and executor
- Single-session transaction support
- CLI and REPL

## Possible Future Work

- Secondary indexes
- More SQL syntax such as `ORDER BY`, `LIMIT`, and joins
- Safer crash recovery through WAL
- Concurrency control
- Better query planning
- `ALTER TABLE`
- More detailed diagnostics
- Import and export tools

These items are not implemented in BaeSQL 0.1.
